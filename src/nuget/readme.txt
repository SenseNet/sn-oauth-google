To finalize the installation and get started with sensenet ECM Google OAuth Provider, please follow these steps:

(install guide is accessible online at: http://community.sensenet.com/docs/install-oauth-google-from-nuget)

1. Build your solution, make sure that there are no build errors.

2. Install sensenet ECM Google OAuth Provider using SnAdmin

   - open a command line and go to the \Admin\bin folder
   - execute the install-oauth-google command with the SnAdmin tool

   .\snadmin install-oauth-google   

You are good to go! Hit F5 and start experimenting!
